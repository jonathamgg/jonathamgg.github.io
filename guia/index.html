Guia

Paso a passo do uso do SARIK.

1º Caso não tenha um app em funcionamento, com a seguinte estrutura:

  • Namespace
  • Services
  • Deployments


2º Você pode usar a aplicação de votação disponível no SARIK. Execute essas próximas etapas para testar o SARIK. Acesse a pasta AppTest/debian e execute os comandos nessa ordem:

$ kubectl create -f namespaces/
 
$ kubectl create -f services/
 
$ kubectl create -f deployements/
