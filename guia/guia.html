Bem-vindo ao guia do SARIK

Abaixo, encontra-se um passo a passo para o download do framework e sua utilização.

DOWNLOAD

O download do SARIK pode ser efeito através do GIT, utilizando a seguinte linha de comando:

git clone git@github.com:jonathamgg/sarik.git

Utilizando o SARIK em uma aplicação

Caso não tenha um app em funcionamento, com a seguinte estrutura:

  • Namespace
  • Services
  • Deployments

Você pode usar a aplicação de votação disponível no SARIK. Acesse a pasta AppTest e escolha a plataforma: minikube/azure/google, acesse e execute os comandos nessa ordem:

$ kubectl create -f namespaces/
$ kubectl create -f services/
$ kubectl create -f deployements/

Aguarde os PODs serem carregados, dependendo da conexão, esse processo demora de 2 até 5 minutos

Retorne para a pasta raiz:

$ cd ../..

Execução do script:

$ bash sarik.sh
